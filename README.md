# AI-DIGITAL-Dashboard Vue Project

> i-screammedia 영어  - 학습결과(대시보드 홈)

### 프레임워크 정보

-   Nodejs : v18.18.2
-   Nuxt3 : v3.8.1
-   Pinia: ^2.1.7
-   Sass: ^1.69.5
-   Vuetify-nuxt-module: ^0.6.6
-   Package Manager : npm

### 설치 및 실행 방법

```bash
# npm
npm install
```

```bash
# npm
npm run dev
```

### 폴더 구성 및 파일 규칙

[폴더 구성]

-   assets : 이미지 및 scss 
-   components : 파일명은 카멜 표기법 사용, 폴더 구성시 소문자 
-   composables : custom 함수 처리, 공통
-   layouts : default
-   pages : 소문자, 동적 라우팅
-   stores : pinia
-   lib : 외부 js 파일
